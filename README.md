**Pixel Art Frame Animation Maker**

1. Input the resolution for the pixel animation you want to create e.g. enter '8' for and 8*8 grid

2. Make a pixel art frame using a pre-selected array of colours

3. Click 'capture frame' when you're happy with it

4. Repeat steps 2 and 3 until you have enough frames for your animation

5. Click 'Animate Frames' to see your animation loop
